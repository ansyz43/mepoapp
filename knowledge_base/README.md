# Knowledge Base

Place `.txt` files in this directory with product information, FAQ, and other knowledge
that the AI assistant will use via RAG (Retrieval Augmented Generation).

## File Format

Each product card should follow this format:

```
### ПРОДУКТ: Product Name
КАТЕГОРИЯ: Category
ОПИСАНИЕ: Description
ПОДХОДИТ ДЛЯ: Use cases (comma-separated)
```

Separate sections with `---` (three dashes on a new line).
